主な機能

Morpholyzerが提供する機能をご紹介します

高精度形態素解析

MeCabエンジンを使用した形態素解析。品詞、活用形、読み方まで詳細に分析します。

視覚的な結果表示

解析結果を見やすい表形式で表示。品詞統計や詳細情報を一目で確認できます。

レスポンシブデザイン

Bootstrap 5を使用したモバイル対応UI。スマートフォンからデスクトップまで快適に利用できます。

API対応

JSON形式でデータを取得可能なRESTful API。外部アプリケーションとの連携も簡単です。

Docker対応

Dockerコンテナでの簡単デプロイメント。環境構築の手間を大幅に削減できます。

高速処理

大量のテキストも高速で解析。リアルタイム処理に対応。

セットアップ方法

簡単3ステップで始められます

Docker環境（推奨）

最も簡単なセットアップ方法です。

# リポジトリをクローン
git clone https://github.com/TakuyaFukumura/morpholyzer.git
cd morpholyzer

# Dockerで起動
docker-compose up web

# ブラウザでアクセス
http://localhost:8000

ローカル環境

Python環境での直接セットアップ。

# MeCabインストール (Ubuntu/Debian)
sudo apt-get install mecab mecab-ipadic-utf8

# Python依存関係
pip install -r requirements.txt

# データベース準備
python manage.py migrate

# サーバー起動
python manage.py runserver
GitHubで詳細を見る

API仕様

プログラムから形態素解析機能を利用できます

POST /api/analyze/

日本語テキストの形態素解析を実行

リクエスト例
curl -X POST http://localhost:8000/api/analyze/ \ -d "text=これは日本語のテストです。"
レスポンス例
{ "success": true, "original_text": "これは日本語のテストです。", "morphemes": [ { "surface": "これ", "pos": "名詞", "pos_detail1": "代名詞", "base_form": "これ", "reading": "コレ", "pronunciation": "コレ" } // ... 他の形態素 ], "pos_summary": { "名詞": 2, "助詞": 2, "助動詞": 1 }, "total_morphemes": 7 }
パラメータ: text (必須, 最大1000文字の日本語テキスト)
